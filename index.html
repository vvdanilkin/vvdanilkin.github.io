Проекты

Тестирование веб- и мобильного приложения Яндекс.Самокат


Дипломный проект по тестированию учебного сервиса по заказу и доставке самокатов в Москве. Сервис состоит из веб-приложения для клиентов и мобильного приложения для курьеров, доставляющих заказнные самокаты. В ходе проекта удалось проработать следующую тестовую документацию:


  • - Mindmap функциональности формы заказа
  • - Чек-лист по требованиям к функциональности "Статус заказа"
  • - Чек-лист валидации полей формы заказа
  • - Тест-кейсы на проверку функциональности мобильного приложения
  • - Чек-лист на проверку API веб-приложения
  • - Баг-репорты
